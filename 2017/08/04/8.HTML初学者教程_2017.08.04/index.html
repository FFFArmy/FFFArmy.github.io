前言

​ 在网上找到一篇教程，闲着无事就翻译一下吧，HTML Beginner Tutorial

​ 让我们开始吧！这个HTML初学者教程假设你之前完全不会HTML和CSS。

​ 如果你仔细阅读每一个页面，那么在接下来的CSS初学者教程之前，你可以很容易地把它所涵盖的内容都学会。

​ 最开始要考虑的事情，最关键的就是HTML用于构成网页而CSS用于表现。HTML只不过是结构化内容，当我们使用CSS时，内容的视觉格式就会出现。你可能会在Web上的其他地方找到不同的教程，但是HTML狗从一开始就专注于最佳实践，从一开始就进入正确的思维方式，最终会带来更好的结果。

属性

图片(lmage)

​ 只是使用存文本格式的话，事情看起来似乎有点乏味和无聊。淡然，网页当然不仅仅是文本，是一个多媒体的内容丰富的作品，最常用的亮点就是图像。

​ <image>标签可以吧图片放入到HTML文档里面，他应该是像这样的：

<img src="http://www.htmldog.com/badge1.gif" width="120" height="90" alt="HTML Dog">

​ src属性告诉浏览器去哪里找图像。像<a>标签，它也可以使用绝对地址，向上面的例子展示的那样，但它通常用相对的地址。比如，你创造了自己的图像并在image文件夹保存为“alienpie.jpg”，那么代码应该是这样的：

<img src="images/alienpie.jpg"...>

​ 需要加上width和height属性，因为如果没有，浏览器会加载图像原始大小，而不是页面加载时设计好的大小，这意味着页面加载时布局可能跳来跳去。

​ <alt>属性是可选的描述。这是一个友好的考虑，为那些看不到图像的用户提供有意义的信息（例如，如果他们无法加载图片）。

​ 值得注意的是，像<br>标签，因为<img>元素没有围绕任何内容，不需要结束标签。

​ 用于当前的HTML的表单的基础标签有<form>、<input>、<textarea>、<select>、<option>

input标签

<input>标签是表单世界的爸爸。它可以采取多种形式，最常见的有以下（在 input reference page看见整个疯狂的家族）：

  • <input type="text">或者简单的<input>是一个标准的文本框。它同样可以有一个属性值，设置初始内容。
  • <input type="password">和文本框相似，但用户输入的内容会被隐藏。
  • <input type="checkbox">是一个复选框，可以由用户打开或者关闭。它同样有一个checked属性（<input type="checkbox" checked>-这个属性不需要值），并打开复选框的初始状态。
  • <input type="radio">和复选框相似，但用户只能在群体中选择一个单选按钮。它同样有checked属性。
  • <input type="submit">是一个按钮，点击会提交表单。你可以用value属性控制提交按钮上的文本，如<input type="submit" value="Ooo. Look. Text on a button. Wow">.
