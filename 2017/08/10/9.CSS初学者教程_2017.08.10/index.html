CSS初学者教程

发表于 2017-08-10 | 分类于 技术 ， 语言 ， CSS

前言

​ 按照惯例，先放原文链接CSS Beginner Tutorial.

​ HTML初学者教程一样，CSS初学者教程假设你什么也不懂(鬼知道英文原文是什么意思)，这个指南是为了教会零基础的人入门。

​ CSS Intermediate Tutorial和 CSS Advanced Tutorial有更加深入的教学。

​ CSS，也叫做层叠样式表(Cascading Styles Sheets，译者注：不重要，反正以后只会叫CSS)，是一种装饰HTML风格和表现形式的方式。HTML只是网页的构成或者是框架内容，样式表是文档的展示。

​ 样式和HTML完全不一样，它们的格式是这样的：property:value并且大部分的property与HTML的标签有关。

目录

选择器(selectors)，性质(Properties)，值(value)-组成CSS的元素

颜色(Color)-如何使用颜色

文本(Text)-如何操作文本框的大小和形状

边缘与填充(Margins and Padding)-如何把东西空间化

边框(Borders)-包围内容的东西

整合在一起(Putting It All Together)-把上述所有东西融为一体。

应用CSS(Applying CSS)

​ 有3种方式在HTML上应用CSS：内联(Inline)、内部应用(internal)、外部应用(external)

Inline

​ Inline样式可以直接放在HTML标签的style属性里面.

​ 他们看起来像这样的：

​ <p style="color: red">text</p>

​ 这会让那个段落的文字变红。

​ 但是，如果您还记得，HTML的最好的实践方法应该是一个独立的、无说明的文档，因此应该尽可能避免内联(Inline)样式。

Internal

​ 样式是内嵌在整个页面的，或者是页面内部的一部分。在<head>标签里面，<style>标签包含了整个页面的所有样式。

​ 这回是的整个页面的段落的文字变红、所有的连接变蓝。
